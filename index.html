Modern, easy to use, fast, open source and multiplatform
STOP-MOTION ANIMATION PROGRAM
What is our goal?
Our goal is to create a very convenient, practical and easy to use graphical interface. This program is suitable for everyone - amateurs and professionals. Keep in mind that this program is stable, but still in heavy development.
